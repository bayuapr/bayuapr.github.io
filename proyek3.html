Pelayanan Terpadu Satu Pintu (PN Sleman)

Proyek ini dibuat dalam waktu kurang lebih 2 bulan, website ini buat untuk mempermudah dalam pelaksanaan pengaduan ke Pengadilan Negeri Sleaman.

Dalam pembuatan sistem Pelayanan Terpadu Satu Pintu ini menggunakan PHP framework Codeigniter 3 sebagai bahasa script yang digunakan untuk membuat halaman website dan MySQL sebagai database tempat penyimpanan data.
